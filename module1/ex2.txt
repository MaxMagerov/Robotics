uptime -p
Показывает, сколько времени компьютер работает, с красиво оформленным выводом

lsof
Показ всех открытых файлов в системе, что полезно для диагностики проблем с блокировкой файлов и управлениями процессами

ss
Это замена netstat, показывающая активные сетевые соединения с более детальной и быстрой информацией о состоянии TCP и UDP соединений. Например, ss -tuln показывает открытые порты

wget -r -l2 -p www.example.com
Загружает веб-страницы рекурсивно, вместе со всеми зависимыми файлами. 

rsync
Используется для синхронизации и резервного копирования файлов с минимальной загрузкой ресурсов. Пример: rsync -avz /src/ /dst/ синхронизирует две директории.

xargs
Позволяет передавать выходные данные одной команды в качестве аргументов для другой. 

df -h
Показ дискового пространства в формате, удобном для чтения. Полезно для быстрой проверки доступного места на дисках и разделах.

alias
Создает псевдонимы для команд. Например, alias ll='ls -lah' задает короткий вызов команды ls с необходимыми параметрами. Очень удобно для сокращения часто используемых команд.

uuidgen
Генерирует уникальные идентификаторы UUID. Используется для создания уникальных имен для файлов, баз данных и других ресурсов.

fuser -v /path/to/file
Показывает, какие процессы используют указанный файл или директорию. Это удобно для выяснения, почему файл не может быть удален или используется другой программой.
